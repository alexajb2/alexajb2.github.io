CupBot RL

A reinforcement learning agent trained to play Cuphead

Project Overview

CupBot is a reinforcement learning agent trained to play the notoriously difficult game Cuphead. The agent uses a combination of deep Q-learning and imitation learning to master the game's complex boss fights.

CupBot in action against a Cuphead boss

Tech Stack

Python PyTorch OpenCV Stable Baselines3 Gymnasium

Key Features

Deep Q-Learning

Implemented a deep Q-network that learns optimal action policies through trial and error, with experience replay to improve stability.

Computer Vision

Used OpenCV for real-time game state detection, including boss positions, projectiles, and player health.

Imitation Learning

Incorporated human gameplay demonstrations to accelerate learning and overcome exploration challenges.

Performance Metrics

Developed custom reward functions and evaluation metrics to track agent improvement over time.

Key achievements:

  • Defeated 5 bosses with a success rate of over 70%
  • Achieved an average survival time of 2 minutes against all bosses
  • Learned complex dodge patterns for multiple attack types

Challenges and Solutions

Developing CupBot presented several unique challenges:

  • Sparse Rewards: Cuphead provides limited feedback on performance. I implemented a custom reward function that considers distance to enemies, projectile avoidance, and damage dealt.
  • Action Space Complexity: The game requires precise timing of multiple actions. I designed a hierarchical action space that combines basic movements into higher-level strategies.
  • Visual Processing: Extracting game state from pixels is challenging. I created a specialized computer vision pipeline that identifies critical game elements with high accuracy.
